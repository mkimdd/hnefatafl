To use, run main.py in the powershell.

Rules this game is based on are from this article:

https://medium.com/boardom/how-to-play-hnefatafl-32d8db70bd16

The program will accept move commands in the follow format:

<source> <destination>

where <source> is in the format: x,y
and <destination> is in the format: x,y

Example:

2,6 3,6
        will move an attacker piece from position (2, 6) to position (3, 6) if that move is allowed
